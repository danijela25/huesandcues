# Hues and Cues — kostur projekta

## Pokretanje servera
```bash
cd server
npm install
npm start
```

## Pokretanje Godot klijenta
1. Otvori `client/project.godot`
2. Pokreni projekat
3. Testiraj `Create Room`
4. U drugom prozoru testiraj `Join Room`

## Sta trenutno radi
- create room
- join room
- lobby
- ready
- start game
- slanje hinta
- klik na grid

## Sledece
- validacija poteza
- drugi hint
- bodovanje
- kraj runde
- game over
